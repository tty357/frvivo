Les Articles Définis et Indéfinis

Une différence importante avec l'espagnol !
En français : le / la / les (définis) et un / une / des (indéfinis).
"La problema" en ES → le problème en FR (masculin !)
  • Je veux une pomme. Quiero una manzana.
  • Le chat est mignon. El gato es lindo.
  • J'aime les fleurs. Me gustan las flores.
🏋️ Exercice : complète
1. J'ai vu chien dans le parc.
2. tour Eiffel est magnifique.
3. Elle mange croissants chaque matin.
4. problème est compliqué.
